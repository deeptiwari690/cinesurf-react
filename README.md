# CineSurf 🎬

Discover something to watch tonight. A movie discovery app built on the TMDB API.

## Features

- 🔍 Search across 900,000+ movies
- 🔥 Trending & popular rows
- ♾️ Infinite-scroll discover feed
- 🎞️ Movie details with cast & trailers
- ⭐ Personal watchlist (sign in required)

## Stack

- **Vite + React 19 + TypeScript**
- **TanStack Query** for server state
- **Zustand** for client state
- **React Router v7** with data loaders
- **Zod** for runtime validation
- **Tailwind CSS v4**
- **Supabase** for auth & watchlist persistence
- **TMDB API** for movie data

## Getting started

```bash
npm install
npm run dev
```
